============
Installation
============


Stable release
--------------

To install standard_bam_processing, run this command in your terminal:

.. code-block:: console

    $ pip install standard_bam_processing

This is the preferred method to install standard_bam_processing, as it will always install the most recent stable release.

If you don't have `pip`_ installed, this `Python installation guide`_ can guide
you through the process.

.. _pip: https://pip.pypa.io
.. _Python installation guide: http://docs.python-guide.org/en/latest/starting/installation/


From sources
------------

The sources for standard_bam_processing can be downloaded from the `Github repo`_.

You can either clone the public repository:

.. code-block:: console

    $ git clone git://github.com/ionox0/standard_bam_processing

Or download the `tarball`_:

.. code-block:: console

    $ curl  -OL https://github.com/ionox0/standard_bam_processing/tarball/master

Once you have a copy of the source, you can install it with:

.. code-block:: console

    $ python setup.py install


.. _Github repo: https://github.com/ionox0/standard_bam_processing
.. _tarball: https://github.com/ionox0/standard_bam_processing/tarball/master
